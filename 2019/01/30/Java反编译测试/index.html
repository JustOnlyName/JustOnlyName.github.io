Java反编译

 我初步理解“编译”是指源代码转换成二进制执行代码的过程，比如将C源代码编译成.exe可执行文件；那么把二进制执行代码转换成源代码的过程就叫“反编译”,意思是将代码、汇编语言这些东西制作成应用程序这个过程的反过程，比如把exe转换为C源代码就叫“反编译”。

那就用一个与应用程序那种可视化界面最接近的我以前的Java程序来测试一下。
