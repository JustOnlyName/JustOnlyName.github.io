Github 搜索技巧

  1. 直接写keyword

  2. in:name(仓库名) keyword

  3. keyword language:python
    in:name keyword language:java

  4. in:description(仓库名下方的作者叙述) keyword

  5. in:readme keyword

  6. keyword stars:>1000

  7. keyword forks:>1000

  8. 日期

    keyword pushed:>2019-02-01(相当于找最近提交更新的)
    keyword pushed:2016-04-30..2016-05-01 日期区间
    keyword created:>=2017-04-30

  9. 排除运算

    keyword pushed:<2017-01-01 -language:java 排除java 语言的结果

  10. 包含搜索

    keyword in:file 搜索文件中含 keyword 的代码
    keyword in:path 搜索路径中包含 keyword
    keyword in:path,file 路径、文件中
    console path:app/public language:javascript 搜索关键字 console ，且语⾔为javascript，在 app/public 下的代码

  11. 主体搜索

    user:USERNAME 用户名搜索
    org: ORGNAME 组织搜索
    repo:USERNAME/REPOSITORY 指定仓库搜索

  12. filename:config.php language:php 搜索文件名 config.php

    mail filename:.properties java 的项目配置文件
